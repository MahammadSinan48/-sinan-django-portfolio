# Sinan Django Portfolio

A modern, responsive, and fully dynamic portfolio website built with Django, PostgreSQL, Tailwind CSS, JavaScript, and GitHub API integration.

## 🚀 Features

- Modern UI/UX
- Mobile Responsive
- Dark & Light Mode
- Animated Hero Section
- Project Showcase
- Skills & Experience
- Blog System
- Certificate Gallery
- GitHub API Integration
- Contact Form
- Admin Dashboard
- SEO Optimized
- PostgreSQL Database
- Secure Authentication

## 🛠 Tech Stack

- Python
- Django
- PostgreSQL
- Tailwind CSS
- JavaScript
- HTML5
- CSS3
- Git & GitHub
- Cloudinary
- Render
